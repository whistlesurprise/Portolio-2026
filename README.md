# Undergraduate Research Portfolio

This repository contains a selection of empirical work completed during my undergraduate studies at the University of Warsaw. It serves as a portfolio of coursework, research projects, and supporting materials relevant to my academic training in economics.

## Repository Structure

### Code
The **Code** directory contains scripts and programs developed for the *Quantitative Economics (2400-ZEWW863)* course at the University of Warsaw. These scripts illustrate the empirical and computational methods used during the course.

### CV
The **CV** directory includes my most recent curriculum vitae.

### Papers
The **Papers** directory contains selected reports and research papers written during my undergraduate studies. These documents reflect my work in applied and empirical economics.
